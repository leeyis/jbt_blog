# jbt_Blog简介

## requirement
- Django==2.0.3
- django-jet==1.0.7
- django-summernote==0.8.8.6
- pytz==2018.3

## 已完成功能：
- 文章管理，包括新增、删除和编辑
- 分类管理，包括新增、删除和编辑
- 标签管理，包括新增、删除和编辑
- 文章概要信息、发布时间、文章分类、标签、浏览次数
- 分页展示
- 点击"阅读全文"显示文章详细内容
- 详情页每刷新一次浏览次数+1
- 文章分类搜索
- 文章标签搜索
- 文章详细页切换上一篇和下一篇文章
- 点击图标回到顶部
- 文章按月归档
